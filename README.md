## Setup

```bash
npm install
npm run dev        # local dev
npm run build      # production build
npm run deploy     # deploy to GitHub Pages
```

> **Note:** This project uses GSAP trial plugins (`gsap-trial`). For production/commercial use, replace with licensed [GSAP Club plugins](https://gsap.com/docs/v3/Installation/).
